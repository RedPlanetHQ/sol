#### General

1. Open command menu `CMD` + `K`
2. Open sigma chat `CMD` + `L`

#### Tasks

1. Create task `CMD` + `N`
2. Schedule `s`
3. Set due date `d`
4. Mark task done `c`
5. Delete tasks `CMD` + `DEL`

#### Navigation

1. Go to Today `G` then `T`
2. Go to Tasks `G` then `M`
3. Go to Lists `G` then `L`
4. Go to Settings `G` then `S`
