## Installation

1. **Download**: Get the latest SOL Desktop App for your platform from the [Releases page.](https://github.com/RedPlanetHQ/sol/releases)
2. **Install & Launch**: Run the installer and open SOL.
3. **Sign Up (Private Beta)**: SOL is currently in Private Beta. Use the invite code provided in your invitation. Follow the setup prompts to create your account or sign in.

## Initial Setup

### Connect Your Accounts

To maximize SOL's potential, connect your core dev tools during onboarding:

- Slack – For message monitoring and task creation
- GitHub – For PR reviews and issue tracking
- Linear – For issue and project management
- Gmail – For email sorting and response drafting

You can add or remove integrations anytime from SOL’s Settings.

### Configure Your Memory

SOL’s C.O.R.E feature powers deep personalization through a secure, independent memory service:

1. In the SOL app, go to the Memory section.
2. Visit core.heysol.ai and create your C.O.R.E account.
3. Generate an API key from your C.O.R.E dashboard.
4. Paste your API key into the SOL Memory section to complete setup.

Note: For more about how and why C.O.R.E works as a separate secure service, see our CORE documentation section.

### Set Up Your First Automation

Automate routine workflows with rules:

1. Open Automations in the sidebar.
2. Click Create New Automation.
3. Describe your automation in plain English.

Tip: The Automations section has best practices and advanced examples.
